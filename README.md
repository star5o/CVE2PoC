# CVE2PoC

**CVE2PoC: LLM-Driven PoC Generation for Web Vulnerabilities from CVE Descriptions**

This repository hosts the benchmark dataset used in our paper on generating web vulnerability PoCs from CVE descriptions with LLMs.

## Repository Structure

- `CVE2PoC-Bench/`: benchmark samples organized by CWE and CVE IDs
- Each CVE case typically includes:
  - `README.md`
  - `docker-compose.yml`
  - `build_depend/`
  - `exp_workspace/`
  - `vuln_info/`
